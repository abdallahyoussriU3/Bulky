## 📂 Project Overview

The **Bulky Book Website** is an **e-commerce platform** where customers can:
- **Browse & view products**
- **Add items to the cart**
- **Checkout using credit card payments**
- **Receive email notifications for orders**

The **Admin Dashboard** allows administrators to:
- **Manage product inventory**
- **Process customer orders**
- **Handle transactions & analytics**

---

## 🛠️ Tech Stack

- **Backend:** ASP.NET Core, C#, Entity Framework Core  
- **Frontend:** Razor Pages, Bootstrap  
- **Database:** Microsoft SQL Server  
- **Security:** Identity Framework, JWT Authentication  
- **Payments:** Stripe API  
- **Deployment:** Azure
