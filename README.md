## 📌 Overview

This project is a **Project Management System** built using **ASP.NET Core Web API**. It allows users to **manage companies, projects, sprints, issues, and comments** while implementing **authentication, authorization, and refresh tokens** for secure access.

This system is designed for **software teams** to efficiently track and manage their work, similar to **Trello, Monday, Linear, and Asana**.

---

## ✨ Features

✅ **Authentication & Authorization** using **JWT** with **Refresh Tokens**  
✅ **Role-based Access Control** (Project Manager, Product Manager, Developer)  
✅ **Company & Project Management**  
✅ **Sprint & Issue Tracking**  
✅ **File & Image Uploads**  
✅ **Exception Handling** via **Middleware & Filters**  
✅ **Repository Pattern with Entity Framework Core**  
✅ **API Versioning & Documentation** (Swagger)  

---

## 🚀 Tech Stack

- **Backend:** ASP.NET Core, C#, Entity Framework Core  
- **Database:** Microsoft SQL Server  
- **Security:** JWT Authentication, Refresh Tokens  
- **Tools:** Postman, Swagger, GitHub
